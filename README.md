# Marine Recorder Selenium Automation
Automate manipulation of records on MR portal with selenium


## ⚠️ Data Deletion Warning

This script **permanently deletes data** from a **Marine Recorder Management** database.

**Do not run this script unless you are certain.**  
If unsure, remove or disable the `delete_replicate` function before execution.

## Overview

Automates the deletion of **replicate projects** from the Marine Recorder Management web application using **Selenium**.
